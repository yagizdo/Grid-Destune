
# Flutter - Grid Destune

Todo App I made with Flutter.

## Run Locally

Clone the project

```bash
  git clone https://github.com/yagizdo/Grid-Todo
```

Go to the project directory

```bash
  cd Grid-Todo
```

Install dependencies

```bash
  flutter pub get
```

Run App

```bash
  flutter run
```
